# UI Reference

This folder stores visual references for future design passes.

- `notion-ai-workspace-reference.pdf`: Notion-inspired white-space, typography, spacing, and restrained color-accent reference.

## Custom Icon Direction

Use the About-section icon as the approved custom SVG direction:

- Notion-inspired illustrated glyphs, not generic line icons.
- Circular color field instead of rounded-square badges.
- Vivid but tasteful background colors, such as Pepsi blue, yellow, red, green, or purple.
- Primary object should use a black outline so it reads clearly at small sizes.
- Inner object can use a second color when it adds meaning, such as Finder-blue folder fill.
- One small accent is allowed when useful, such as a yellow caution mark, red exclamation, or status dot.
- Avoid literal connector lines, decorative tracks, and extra UI framing unless the section specifically needs them.
- Icons should feel useful and a little playful, but not loud or cartoonish.
