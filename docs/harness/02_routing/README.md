# 02 Routing

Owner: Cerebral router.

Allowed inputs:

```text
intake note
raw user request
Opportunity HQ candidate
command idea
proof/workflow/offer/job lead
```

Allowed outputs:

```text
[lane] Cash Jobs | Career Jobs | Freelance | Offer | Proof
[owner] durable owner
[intent] one plain sentence
[shape] capture | task | sub-task | dependency | proof | command idea | operating rule
[tools] skill stack and action surface
[review] review gate
[next] next concrete action
[do-not] blocked surfaces or actions
```

Do not:

```text
create new lanes
create folders/tags/commands
mutate Notion
rename surfaces
route around Opportunity HQ for durable task state
```

Next action rule: if it has task weight, send it to `03_task-shape`; if not, keep it in Bear or docs.
