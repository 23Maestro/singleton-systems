# Personal Ops Google Handoff

Start read-only. Report the current Google files, Apps Script projects, and
owner map before creating or changing data.

## Owners

```text
Google Sheets / Forms / Apps Script -> Personal Ops source data
Linear                              -> active decision or system work
Next/Vercel                         -> review only
```

The dashboard may link to a Google surface but must not copy its status or write
without an explicit owner action.

## Chores: Smallest Useful Shape

The first slice is a Google Form plus Sheet. The only durable chore fields are:

```text
Room
Task
```

Allowed rooms: `Office`, `Bathroom`, `Laundry`, `Auto`, `Kitchen`, `Living
Room`, and `Garage`. Keep the form-to-sheet route literal. Do not add priority,
status, dates, assignees, notes, categories, or recurring rules until the small
list proves they are necessary.

## Finance: Separate Decision

Do not mix finance into the chores build. First report the current `23 Space`
finance source, inputs, outputs, duplicates, and a proposed review surface.
Create a separate Linear decision before modifying finance data.

## Verification

Before mutation: show the source map and proposed fields. After mutation:
verify one form submission, one Sheet row, and one review link. Do not populate
real finance data during a chores test.
