# 06 Memory Compaction

Owner: docs/skills memory summary.

Allowed inputs:

```text
verified decision
changed files/surfaces
task shape
proof links
blockers/dependencies
remaining next action
```

Allowed outputs:

```text
[memory] decision
[owner] owner surface
[lane] canonical lane/name
[shape] task shape
[block] blockers/dependencies
[evidence] proof links or verification result
[next] remaining next action
```

Do not keep:

```text
repeated phrasing
side chatter
dead options
unapproved automation ideas
raw transcript bulk
```

Do not:

```text
store unreviewed automation ideas as commitments
summarize raw noise as durable truth
create memory for one-off phrasing
hide the remaining next action
```

Next action rule: compact to the next useful action, not a full transcript.
