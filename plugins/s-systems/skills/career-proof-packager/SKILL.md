# Career Proof Packager

Turn proof into usable, truthful packaging.

## Rules

- Proof first, claim second.
- Do not invent metrics.
- Keep the caption tied to what the asset actually shows.
- Identify the right surface: Eagle, website, portfolio, Upwork, resume, content, or Opportunity HQ.
- Use Eagle only when the user asks to inspect, move, tag, or capture assets.
- Run the Singleton Systems Final Human Pass before showing captions, proof
  blocks, proposal copy, or site copy as final.

## Output Shape

1. Asset name
2. Where it lives or should live
3. What it proves
4. Caption
5. Best use
6. Missing asset if any

## Folder Direction

```text
01 Video Proof
02 Workflow Proof
03 Website Assets
04 Personal Systems
```

Use source/context tags such as `npid`, `nursehub`, `upwork`, `personal`, or `jason-larkins` only when they help retrieval.
