# Cover Letter Casual

Write like a capable operator sending a real note.

## Style

- Short.
- Human.
- Specific to the role or client.
- No corporate filler.
- No "I am excited to apply" unless requested.
- No fake certainty or invented proof.
- Run the Singleton Systems Final Human Pass before showing the note as final.

## Structure

1. Plain opener
2. Why this maps to the user's experience
3. One proof point or concrete example
4. Simple close

## Required Output

Provide:

1. Direct version
2. Warmer version
3. Very short version

Keep each version tight unless the user asks for a fuller letter.
