# Cerebral Router

Use before work crosses Linear, GitHub, Supabase, Opportunity HQ, Eagle,
Raycast, Shortcuts, Next/Vercel, docs, or LikeC4.

## Owner Map

```text
unselected thought -> Linear Command + Ideas
active system plan -> Linear Singleton Systems
fuzzy or branching build -> GitHub Wayfinder
queryable event or dashboard fact -> Supabase
career workflow state -> Opportunity HQ
proof asset -> Eagle
desktop/mobile action -> Raycast or Shortcuts
active-week review -> Next/Vercel
reviewed architecture -> LikeC4
```

## Cerebral Tags

```text
[owner] one durable surface
[intent] one sentence
[tools] smallest stack
[review] mutation gate
[verify] proof command or readback
```

`23M-89` owns the Linear GraphQL gateway for Codex, Raycast, and Shortcuts.
Keep its credentials server-side.

## Route

1. Name one owner.
2. Select the smallest skill or tool stack.
3. State the review gate before mutation.
4. State the check that proves the result.

Do not copy task state into the dashboard or full ticket/template text into
Supabase. Keep credentials server-side.
The dashboard reads owner state and opens owner links.
